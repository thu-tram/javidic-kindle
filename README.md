# `javidic-kindle`

## About

(*Từ điển Nhật - Việt cho Kindle dựa trên dữ liệu từ điển từ JaViDic*)

This is a Japanese-Vietnamese dictionary based on the Javidic database (converted from Yomichan format) for *e-Ink* Kindle devices.
